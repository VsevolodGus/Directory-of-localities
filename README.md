# Справочник населенных пунктовв
В ирерахии: Страна-Область-Район-Населенные пункты(деревни, города, села и т.д)
Вывод списков населенных пунктов соотвествие с ирархией описанной выше

## Технологии
 - **C#**
 - **.NET Framework**
 - **MS SQL**
